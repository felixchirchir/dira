Hotspot Billing

Our hotspot billing system provides a seamless way to monetise your Wi-Fi network. It offers easy setup, flexible pricing options, and real-time usage tracking for public spaces, cafes, hotels, and more.

Key features:

  • User-friendly captive portal
  • Multiple payment options including M-Pesa
  • Customisable bandwidth and time-based packages
  • Detailed usage reports and analytics

Requirements for automation:

  • Mikrotik router with hotspot enabled
  • Stable internet connection
  • Diralink billing software
  • M-Pesa API integration
  • User management system

PPPoE

Our PPPoE (Point-to-Point Protocol over Ethernet) solution offers a robust and secure method for delivering broadband internet services. It's ideal for ISPs and enterprises looking to provide reliable, individual connections to multiple users.

Key features:

  • Secure, individual user authentication
  • Flexible bandwidth allocation
  • Easy scalability for growing networks
  • Detailed connection logs and monitoring

Requirements for automation:

  • Mikrotik router with PPPoE server configured
  • Diralink billing software
  • RADIUS server for authentication
  • Automated provisioning system
  • Bandwidth management tools
