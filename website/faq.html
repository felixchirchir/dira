Frequently Asked Questions

What is Dira Cloud, and how does it manage billing for hotspot and PPPoE services?

Dira Cloud is a company that offers cloud-based billing solutions specifically designed for Internet Service Providers (ISPs) using MikroTik routers. It provides a billing system that integrates seamlessly with MikroTik routers to manage and automate billing for both hotspot and PPPoE services. The system allows ISPs to track customer usage, generate bills, and handle payments efficiently for both prepaid and postpaid models.

How does Dira Cloud's billing system integrate with a MikroTik router for hotspot services?

Dira Cloud's billing system integrates with a MikroTik router by configuring the router to handle user authentication and authorization through a captive portal. When a user connects to the hotspot, they are redirected to the billing system where they can log in, purchase a data package, or pay for usage. The system tracks their session duration or data usage, generates an invoice, and can automatically disconnect users when their balance is exhausted.

How does the billing system handle PPPoE billing?

For PPPoE users, the Dira Cloud billing system integrates with the MikroTik router to manage user authentication and bandwidth limits. Customers authenticate via their PPPoE credentials, and the billing system monitors their usage (such as bandwidth or session time). It supports prepaid and postpaid models, automatically generating bills based on usage, or setting fixed monthly charges. The system can also send automated notifications when users are approaching their limits or when a payment is due.

What are the key requirements for implementing Dira Cloud's billing system using a MikroTik router?

To implement Dira Cloud's billing system with a MikroTik router, the following requirements are essential:

  • MikroTik Router: A properly configured MikroTik router capable of managing hotspot or PPPoE connections.
  • Internet Connection: A reliable internet connection to ensure smooth communication between the MikroTik router and the Dira Cloud billing system.
  • Dira Cloud Account: An active subscription to Dira Cloud's billing platform to access and configure billing features.
  • RouterOS Compatibility: The MikroTik router must be running a compatible version of RouterOS that supports hotspot and PPPoE configurations.
  • RADIUS Server Configuration: For PPPoE billing, a RADIUS (Remote Authentication Dial-In User Service) server may be required to handle authentication and accounting, which can be integrated with the Dira Cloud billing system.
  • Payment Gateway Integration: Integration with payment gateways (such as PayPal or mobile money services) to allow for automatic or manual payments by customers.

How does Dira Cloud support both prepaid and postpaid billing models for ISPs?

Dira Cloud offers flexibility in billing by supporting both prepaid and postpaid models:

  • Prepaid Billing: Customers purchase data packages or time-based internet access in advance. Once the purchased quota is used, the system can automatically restrict or disconnect access until the user purchases more.
  • Postpaid Billing: Users are billed at the end of the billing period (monthly, for example) based on their usage. The system generates detailed invoices and sends them to customers for payment before the due date.

What are the benefits of using Dira Cloud's billing system for hotspot and PPPoE services?

The key benefits include:

  • Automation: The system automates the billing process, reducing the manual effort required to track customer usage, send invoices, and collect payments.
  • Scalability: Dira Cloud's system can scale to handle a large number of hotspot and PPPoE users, making it suitable for ISPs of various sizes.
  • Payment Flexibility: The system integrates with multiple payment gateways, allowing users to pay with various methods, such as credit cards, mobile money, or bank transfers.
  • Usage Monitoring: ISPs can easily monitor customer usage, bandwidth consumption, and connection time in real time, allowing for better customer management and service optimization.
  • Security: It provides secure handling of user authentication and billing data, ensuring that sensitive information is protected.
  • Notifications and Alerts: The system can send automated notifications for low balance, upcoming due dates, or service expiration, improving the customer experience.

How does Dira Cloud ensure reliable service continuity when integrated with MikroTik routers?

Dira Cloud ensures reliable service continuity by:

  • Cloud-Based Architecture: The billing system operates in the cloud, reducing downtime and dependency on local servers. This allows continuous service availability.
  • Failover and Redundancy: Dira Cloud's infrastructure includes failover mechanisms to maintain uptime even during technical failures.
  • Automatic Service Suspension/Restoration: The system can automatically suspend services if a customer's balance runs out or a payment is overdue, and can restore access once payment is received.

Can Dira Cloud generate detailed usage reports for hotspot and PPPoE users?

Yes, Dira Cloud can generate detailed usage reports for both hotspot and PPPoE users. The system provides insights such as:

  • Total Data Usage: Breakdown of data consumed by individual users over a specified period.
  • Session Information: Start and end times of user sessions, duration of usage, and bandwidth consumed.
  • Billing Summary: Invoices, payments made, and outstanding balances.
  • Customer Insights: Detailed reports on customer behavior, such as frequent usage patterns, helping ISPs make informed decisions.
